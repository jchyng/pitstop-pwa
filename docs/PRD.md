# Pitstop PWA — 제품 기획서

## 핵심 한 줄

차종 선택 → 소모품 정비 주기 즉시 확인 + 교환 시기 웹 푸시 알림

## 타겟 유저

차량 관리에 신경은 쓰지만, 어떤 소모품을 언제 갈아야 하는지 매번 검색하기 귀찮은 사람.

## 개발 단계

| Phase | 기능                                           | 인증          | 저장소                         |
| ----- | ---------------------------------------------- | ------------- | ------------------------------ |
| 1     | 차종 선택 + 주행거리 입력 + 소모품 긴급도 조회 | 없음          | 정적 JSON + localStorage       |
| 2     | 정비 기록 저장 + 기기 간 동기화                | 카카오 로그인 | Supabase PostgreSQL            |
| 3     | 교환 임박 웹 푸시 알림                         | Phase 2 유지  | Supabase Edge Functions (cron) |

## 지원 차종

### 지인 테스트 단계 (현재)

| 차종 | car_id |
|------|--------|
| 아반떼 MD 가솔린 | `avante-md-gasoline` |
| 그랜저 HG 2.4 가솔린 | `grandeur-hg-gasoline` |
| 렉스턴 스포츠 칸 디젤 | `rexton-sports-khan-diesel` |

지인 피드백 수렴 후 차종 추가 여부 결정. 현재 `public/cars/` 에는 40종 JSON이 준비돼 있으며, `index.json`에 추가하는 것만으로 서비스에 노출된다.

## 핵심 화면

### Phase 1

- **메인 대시보드** (`/`): 차종 선택 + 주행거리 입력 + 소모품 목록 (긴급도 순 정렬)
  - 뷰 토글: **봐야 할 항목** (과기한+위급만, 배지 카운트) / **전체보기** (카테고리 섹션별)

### Phase 2

- **정비 기록 입력**: 날짜 + 주행거리 + 항목 선택 + 메모(선택)
- **서비스 이력** (`/history`): 월별 타임라인 + 카테고리 필터 칩
- **차량 선택**: 차량 추가/전환

### 소모품 상태 (Status)

| 상태       | 조건          | 카드 우측 표시                  |
| ---------- | ------------- | ------------------------------- |
| 과기한     | ratio ≤ 0     | `N km 초과` (또는 `N일 초과`)   |
| 위급       | 0 < ratio ≤ 1 | `N km 남음` (또는 `N개월 남음`) |
| 정상       | ratio > 1     | `N km 남음` (또는 `N개월 남음`) |
| 알 수 없음 | 계산 불가     | `— / 미기록`                    |

stat 단위: km 기준이 있으면 km, 시간 기준만 있으면 개월/일 표시

### 소모품 카테고리 (전체보기 섹션 순서)

| 섹션           | 항목                                                                           |
| -------------- | ------------------------------------------------------------------------------ |
| 엔진·오일      | 엔진오일, 오일필터, 에어크리너 필터                                            |
| 연료·증발가스  | 연료필터, 증발가스 계통                                                        |
| 공조·외부      | 에어컨필터, 와이퍼 블레이드                                                    |
| 제동·냉각·변속 | 브레이크액, 브레이크패드(전방), 브레이크패드(후방), 냉각수(부동액), 변속기오일 |
| 점화·벨트      | 점화플러그 배선·코일, 점화플러그(이리듐), 구동벨트(타이밍벨트)                 |
| 타이어·배터리  | 타이어, 배터리                                                                 |

### 정비 이력 카테고리 필터

`전체` / `🛢️ 엔진오일` / `🔄 타이어` / `🛑 브레이크` / `💨 에어필터`

## 성공 기준

- 차종 선택 후 10초 안에 전체 소모품 정비 주기 확인 가능
- 주행거리 입력 → 교환 필요한 소모품이 상단에 빨간색으로 표시
- 카카오 로그인 후 정비 기록이 다른 기기에서도 보임
- 웹 푸시 알림이 백그라운드에서 도착함

## 배포

- 플랫폼: Vercel (Next.js, 무료 티어)
- PWA: `next-pwa` 패키지 (서비스 워커 + 매니페스트)
- CI/CD: Vercel Git 연동
