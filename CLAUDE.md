# Pitstop PWA

한국 차량 소모품 정비 주기 알리미 PWA.

## 트리거 → 읽을 파일

| 상황                      | 읽을 파일                                                 |
| ------------------------- | --------------------------------------------------------- |
| 새 컴포넌트·화면 작업     | `docs/DESIGN.md` + `design-reference/main-dashboard.html` |
| 데이터 구조·긴급도 로직   | `docs/ARCHITECTURE.md`                                    |
| Phase 범위·화면 목록 확인 | `docs/PRD.md`                                             |
| 작업 완료                 | `.claude/rules/checklist.md` → `/commit` 커맨드로 커밋     |
| 코드 작성 원칙            | `.claude/rules/code.md`                                   |
